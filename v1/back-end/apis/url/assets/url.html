URL Shortener API Docs

User Stories:

  1. You submit a URL as a parameter and receive a shortened URL in JSON response.
    API endpoint is GET https://brusbilis.com/freecodecamp/v1/apis/url/v1/validUrl
  2. If you pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
  3. If you visit a shortened URL you will be redirected to the original link.

Example Usage:

  • Show all shortened URLs
    GET https://brusbilis.com/freecodecamp/v1/apis/url/v1/list
  • Create a new shortened URL
    GET https://brusbilis.com/freecodecamp/v1/apis/url/v1/new/http://freecodecamp.com
  • Use a shortened URL
    GET https://brusbilis.com/freecodecamp/v1/apis/url/v1/168

Example Output:

{
  "original":"https://brusbilis.com/freecodecamp/v1/apis/url/v1/new/http://freecodecamp.com",
  "short": "https://brusbilis.com/freecodecamp/v1/apis/url/v1/168" 
}
