Exercise Tracker API Docs

User Stories:

  1. You can create a user by posting form data username to /tracker/v1/newUser and returned will be an object with username and _id.
  2. You can get all users /tracker/v1/users.
  3. You can add an exercise to any user by posting form data username, description, duration. /tracker/v1/add no date supplied it will use current date.
  4. You can retrieve a full exercise log of any user by getting /tracker/api/log with a parameter of username.
  5. You can retrieve part of the log of any user by also passing optional parameters.&from= &to= use date format yyyy-mm-dd and &limit= is a integer

Example Usage:

  • Show all users
    GET https://brusbilis.com/freecodecamp/5-api/tracker/v1/users
  • Get users exercise log
    GET https://brusbilis.com/freecodecamp/5-api/tracker/v1/log?user=user
    GET https://brusbilis.com/freecodecamp/5-api/tracker/v1/log?user=user&from=2016-10-10&to=2017-01-01&limit=100
  • Create a new user
    POST /tracker/v1/newUser
  • Add exercises
    POST /tracker/v1/add
